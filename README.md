## Decade별 영화 장르 분포 분석

영화 장르 분포를 10년 단위로 구분하여 체계적으로 검토한 결과, 각 시대별 장르의 부상과 쇠퇴는 단순한 산업적 선택이 아니라, **동시대 사회‧정치‧경제적 맥락과 긴밀히 연결**되어 있었다.
이는 기존 COO 연구가 소비자 인식이 사회적 맥락에 따라 달라진다고 지적한 것과 같은 궤를 공유한다 (Nagashima 1970; Balabanis & Diamantopoulos 2004).

* **1930년대 대공황 시기**: 코미디와 뮤지컬 장르가 상위권 → 불황 속 대중이 일상의 고단함에서 벗어나려는 심리와 직접 연결
* **1940년대**: 드라마와 스릴러 장르 증가 → 전시(戰時)와 전후 사회 불안이 영화적 재현으로 표출
* **1980년대 이후**: 액션과 어드벤처 강세 → 글로벌 블록버스터 산업 발달과 CGI 등 기술 혁신의 영향

이러한 결과는 소비자 인식 연구와 마찬가지로, 영화 장르 소비 또한 단일 요인으로 설명되지 않고, **사회적·정치적·경제적·기술적 환경의 상호작용**으로 형성됨을 시사한다.
즉, 특정 시대의 장르 Top3는 단순한 인기 통계가 아니라, **동시대 대중이 무엇을 갈망하고 두려워했는지를 드러내는 문화적 지표**로 볼 수 있다.

## 요약 고찰

위 표에서 확인되듯, 각 시대별 장르 분포는 단순한 영화 제작 경향을 넘어 **사회·정치·경제적 맥락과 긴밀히 연결**된다.

* 1930년대: 코미디와 뮤지컬 → 불황기의 탈출구 역할
* 1970년대: 스릴러와 범죄 → 사회 불신 심화 반영
* 2010년대~2020년대: 액션·슈퍼히어로물·스릴러 → 글로벌화와 팬데믹 시대 대중 심리 반영

이러한 양상은 COO/COP 연구와 마찬가지로, 영화 장르 역시 **동시대적 환경과 대중의 정서적 수요가 결합**하여 형성된다는 점을 시사한다.
